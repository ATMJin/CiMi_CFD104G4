* header
* 請使用header標籤，不要使用 div class="header"
* 網頁頭註解請放在header標籤上方
* 全所有頁面的header都應該相同，除了辨別該頁的特效
* main
* 最外層請使用main標籤，不要使用 div class="container"
* 網頁內容註解請放在main標籤上方，且與header間格2個空白行

* ===== 文件路徑放置規則 ===== *
* 請不準、不要、不可以使用中文命名資料夾及檔案
* 所有路徑請使用相對路徑抓取
* html檔請不要放置在其他子資料夾內
* js檔請放置在 js資料夾內
* css檔請放置在 assets/css資料夾內
* 圖片檔請放置在 assets/images資料夾內
* 圖片檔命名格式： 使用頁面_使用區塊(class)_icon/bg/X.副檔名
* 影片、影音檔請放置在 assets/media資料夾內
* PHP檔請放置在 phps資料夾內

* ===== 標籤、id、class命名規則 ===== *
* 請不準、不要、不可以使用中文命名class、id
* 所有的複合字請使用 _ 串接 e.g. class="many_words"
* 不需要使用BEM
* id、class="請使用雙引號包"
* 容器與內容分開，除了以下容器以外，容器後面加上 _box
- section標籤
- article標籤
- container
- row
* 自行判斷使用section或div * section寬度固定100%
* 每個section內最少要有一個容器，可用來控制寬度(padding)
* 卡片式排版，容器請以 item_box 命名，卡片以 item 命名
* 圖片容器請固定使用 img_box
* 文字容器請固定使用 txt_box
* 清單請使用 ul/ol li 標籤

* ===== 註解及空行規則 ===== *
* 請大家註解寫中文
* 能註解盡量註解 * tab縮排請設定為4個空格
* 請註明每個一區塊是做什麼的
* 新的區塊註解請與上面內容間格1個空白行 [1]
* section中內容開始前與結束後請各留1個空白行 [3]
* 每個section間請留1個空白行 [4]

* ===== css選取器規則 ===== *
* 請使用 reset.css
* 一開始先都 * {box-sizing: border-box;}，有需要再用contain-box(在reset.css裡面有了)
* css選取器撰寫規則如下：
1. 先寫 header / main / footer
2. 寫區塊 section.XXX
e.g. main section.news .item:nth-child(2) .img_box img {...}

* footer
* 請使用footer標籤，不要使用 div class="footer"
* 網頁尾註解請放在footer標籤上方，並切齊body標籤，且與main間格2個空白行
* 所有頁面的footer都應該相同，除了辨別該頁的特效
